Id Structure

Ids may be generated with the type of Id, which is a Value Class, or as Longs. The value class contains some methods which extract the parameter of the id. Each id is composed of:

  • A timestamp with millisecond precision and a custom epoch.
  • A node id in order to produce unique ids on a multi-server environment.
  • A sequence number to avoid collisions on the same millisecond.

The number of bits devodeted to each of the parameters and the epoch is defined by a Scheme.
